# Upgrading to Hooka v1

This guide covers the runtime and schema expectations for the stable `1.0.0` line.

## Required env changes

Server deployments now require:

- `HOOKA_WEBHOOK_SECRET`
- `HOOKA_ADMIN_TOKEN`

Optional but newly supported server/runtime controls include:

- `HOOKA_CORS_ORIGINS`
- `HOOKA_MAX_BODY_BYTES`
- `HOOKA_RATE_LIMIT_GLOBAL_API_LIMIT`
- `HOOKA_RATE_LIMIT_GLOBAL_WEBHOOK_LIMIT`
- `HOOKA_RETENTION_RUN_DAYS`
- `HOOKA_RETENTION_AUDIT_DAYS`
- `HOOKA_RETENTION_SWEEP_INTERVAL_HOURS`

Hooka also resolves targets from:

- `HOOKA_TARGETS_PATH`
- default: `.hooka/targets.json`

## Runtime changes

- read/admin APIs are protected by `Authorization: Bearer <HOOKA_ADMIN_TOKEN>`
- `/api/targets` and `/api/targets/:id` expose configured targets
- `POST /api/targets`, `PUT /api/targets/:id`, and `DELETE /api/targets/:id` manage file-backed targets
- `GET /api/audit-events` exposes security, policy, and target mutation events
- `/api/runs/:id/retry` retries terminal runs by enqueueing a new run
- `POST /api/events/ticket` issues short-lived SSE tickets
- `/api/events/stream` emits SSE updates for run events and worker heartbeats when called with a valid ticket
- `GET /api/openapi.json` exposes the machine-readable API document
- target-based webhooks are supported alongside existing generic `taskId + input` webhooks
- `hooka cleanup` prunes old terminal runs and audit events

## Run model changes

Hooka now records additional run metadata including:

- `targetId`
- `maxAttempts`
- `nextRetryAt`
- `lastErrorCode`

Supported statuses now include:

- `dead-lettered`

Retryable failures are requeued with backoff. Exhausted retry budget or recovered stuck runs can end in `dead-lettered`.

## Compatibility notes

- existing generic webhook payloads remain valid
- existing WordPress compatibility webhook route remains valid
- worker image taxonomy and preset aliases are unchanged
- official support remains single-node self-host with SQLite
- `?token=` SSE query auth is no longer supported; use the ticket flow instead
